# KBO Historical Mode for OOTP

[한국어](./README.md) | **English**

This is the release package for the KBO Historical Mode project. It bundles the database, uniforms, logos, facegen assets, ballparks, and stats resources used for OOTP.

## Included

- `database`
- `logos`
- `uniforms`
- `facegen`
- `ballparks`
- `stats`
- `tools`

## Installation

### Basic Setup

1. Extract or copy this folder to any working location.
2. Use the contents of this package when creating a new OOTP game or setting up a related league.
3. Copy the contents of `database`, `logos`, `uniforms`, `facegen`, `ballparks`, and `stats` into your league or mod folder as needed.
4. Back up existing files before overwriting anything.

### Recommended

- Starting from a fresh save is recommended.
- After installation, verify that logos, uniforms, ballparks, and facegen assets are linked correctly.
- Back up original files if you may want to restore the default setup later.
